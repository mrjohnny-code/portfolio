# Frontend Developer Portfolio

Личный сайт-портфолио frontend-разработчика с краткой информацией о навыках и примерами реализованных проектов. Создан с использованием AI-инструментов.

## 🔗 Live Demo

https://mrjohnny-code.github.io/portfolio/

## 📌 О проекте

Сайт создан для демонстрации опыта frontend-разработки, реализованных проектов и используемых технологий.

В портфолио представлены:
- коммерческие проекты
- учебные работы
- тестовые задания

Проект оптимизирован по скорости загрузки и адаптирован для различных устройств.

## ⚙️ Стек технологий

- React
- JavaScript (ES6+)
- SCSS
- Vite
- PhotoSwipe (галерея изображений)

## 🚀 Возможности

- фильтрация проектов по категориям
- lazy-loading проектов (Load More)
- lightbox для просмотра изображений
- адаптивная верстка
- оптимизация загрузки изображений
